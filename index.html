How VidarDB Works?

VidarDB can do everything PostgresSQL can do, but with our groundbreaking technology, we make VidarDB run like a speedy antelope.

Blazingly Fast Querying

Just throw your data to VidarDB as frequent as you want, since VidarDB is 50X faster in data ingestion; If you query the recent data, VidarDB offers up to 1000X speedup. Unlike general operational database, VidarDB is designed for modern memory architecture. Fresh data sit in RAM first, and will be automatically flushed to secondary memory when RAM is full.

Responsive Data Analysis
